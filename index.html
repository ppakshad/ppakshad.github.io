An In-Depth Analysis of a Cyber Attack: Case Study and Security Insights

Puya Pakshad
IGI Global Scientific Publishing

This paper presents an in-depth analysis of the DarkSeoul incident, detailing the techniques involved and offering insights and recommendations for the global cybersecurity community. The motivations and strategic objectives behind the attack are examined, as well as South Korea's response and the broader implications for cybersecurity policy. Our analysis highlights key vulnerabilities and underscores the urgency for enhanced defenses against state sponsored cyber threats.
This paper has been published as a book chapter in the book titled Artificial Intelligence in Cybersecurity and Forensic Practices.

A security vulnerability predictor based on source code metrics

Puya Pakshad, Alireza Shameli-Sendi, Behzad Khalaji Emamzadeh Abbasi
Springer, Journal of Computer Virology and Hacking Techniques

Detecting security vulnerabilities in software source code is a key challenge in software security. This study focuses on three objectives: (1) identifying vulnerable functions in C/C++ programs using code metrics to reduce security testing costs, (2) determining the type of attack related to each vulnerability, and (3) analyzing the relationship between code metrics and vulnerabilities to understand which code structures are prone to flaws. Using graph-based static analysis, source code metrics were extracted, and a machine learning model was developed for vulnerability prediction.
